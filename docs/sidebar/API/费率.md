# Rate - 获取服务价格

<table><tr><td bgcolor=#bdbdbd>POST</td><td>/v1/open-api/rate</td></tr></table>


### 请求头部

|   标签    | 必填	 |                         说明	                          |
| :--------: | :------: | :----------------------------------------------------------: |
|  Api-Key   |   ✓     |  Api Key，请登录系统，在我的个人资料中找到|
| Api-Secret |   ✓     | Api Secret，请登录系统，在我的个人资料中找到 |

| ﹂﹂addresses              |  [array]  |          | 地址数组                                                  |
| ﹂﹂city                   | [string]  |    √     | 城市                                                            |
| ﹂﹂name                   | [string]  |          | 收件公司名或收件人名                 |
